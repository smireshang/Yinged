# Yinged

Yinged (二改自[Ying](https://github.com/Siooooooooo/Ying))是一个 **Typecho 简约风主题**，主打轻量、清爽、阅读优先。

## 主题概览

- 适配 Typecho 博客系统。
- 页面结构简洁，突出文章内容。
- 提供轻量的页面切换过渡与加载反馈。
- 默认不包含评论系统（已移除评论模板与样式）。

## 当前功能特点

### 1) 头部与导航
- 站点头像、标题与一句话简介展示。
- 顶部导航支持首页与独立页面入口。
- 导航右侧内置搜索按钮（放大镜），点击展开搜索框进行站内检索。

### 2) 页面切换与加载反馈
- 内部链接点击后显示顶部 loading 动图反馈。
- 内容区域（头部以下）采用轻量“由上到下依次渐入”过渡效果，避免拖沓。

### 3) 文章与图片体验
- 正文图片支持懒加载：
- 图片支持灯箱预览

### 4) 基础阅读能力
- 首页文章列表展示。
- 归档页模板。
- 404 页面与搜索入口。
- 回到顶部按钮。

## 文件结构（核心）

- `header.php`：站点头部、导航、搜索入口、顶部 loading 容器。
- `footer.php`：页脚、脚本引入、回到顶部按钮。
- `index.php`：首页列表模板。
- `post.php`：文章详情模板。
- `page.php`：独立页面模板。
- `articles.php`：归档页模板。
- `js/app.js`：导航搜索、页面过渡、图片懒加载、灯箱、回顶交互。
- `css/main.css`：主题主样式。

## 使用说明

1. 将主题放入 Typecho 的主题目录并启用。
2. 在主题设置中配置头像、一言、备案号等信息。
3. 建议文章图片设置 `title` 属性，以获得更完整的图注体验。

## 说明

- 本主题保持“简约优先”路线，适合偏内容展示的个人博客。
- 若你需要评论能力，可自行接入外部评论系统（如静态评论方案）。
